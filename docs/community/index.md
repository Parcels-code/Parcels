# Community

Parcels users and developers interact in a vibrant community: if you are doing any kind of Lagrangian modelling and/or analysis check out the [CLAM community on Zulip](https://clam-community.github.io/). If you need more help with Parcels, try the [Discussions page on GitHub](https://github.com/Parcels-code/parcels/discussions). If you think you found a bug, file an [Issue on GitHub](https://github.com/Parcels-code/parcels/issues). If you want to help improve Parcels, see the [Contributing](../contributing/index.md) page.
